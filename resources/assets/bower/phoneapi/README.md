# phoneapi

install
=====

```bower
  bower install phoneapi
```

depencies
-----
* vuejs: ~1.0.26
* pusher: ^3.0.0.
* jquery: 2.2

usage
=====
```php
  <div class="phone-app">
    <? include 'bower/phoneapi/dist/template/_phone_api.php'; ?>
    <phone user_id="102" type="egerep" key="pusher_key"></phone>
  </div>
```
